Thank you for wanting to improve this site!

Errors or issues with the site can be reported either via Reddit or Github:
  • Reddit Private Message
  • Github Issues Page
If you wish to have a deck showcased as an example in the Deck Viewer landing page, then message the deck code and details to either:
  • Kollieflower on Reddit
  • DrQuint on Reddit
  • Find us on Valve's official Discord server and message us from there
There are ultimately no definite criteria for what decks we'll show as examples, nor schedule for when we'll rotate decks out, so anyone is free to send.

Thanks!
